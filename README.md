# terse-ui

## document
[terse-docs](https://ziguangliunian.github.io/terse-docs/)
## install
yarn add terse-ui
## Quick Start
```
import Vue from 'vue'
import App from './App.vue'
import terse from 'terse'
import 'terse/lib/terse-ui.css'

Vue.use(terse)
new Vue({
  render: h => h(App),
}).$mount('#app')

```
